# Mini site Noël – Propositions de théâtre

Ce mini-site statique fonctionne sans build (HTML/CSS + images SVG).

## Déploiement sur GitHub Pages
1. Crée un repo GitHub (ex: `noel-theatre`).
2. Glisse-dépose tout le contenu de ce dossier à la racine du repo (ou pousse via git).
3. Dans GitHub : **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: `main` / folder `/ (root)`
4. Attends la publication : ton site sera accessible à l’URL GitHub Pages fournie.

## Modifier les propositions
- Les pages sont : `marianne.html`, `louna.html`, `adel.html`
- Les miniatures (SVG) sont dans `assets/thumbs/`
- Les avatars sont dans `assets/avatars/`
